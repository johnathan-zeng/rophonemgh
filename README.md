# Hospital Phone Trees

This repo contains lightweight mobile-friendly phone tree pages for:

- Massachusetts General Hospital: [mgh.html](./mgh.html)
- Brigham and Women's Hospital / DFCI: [bwh.html](./bwh.html)
- MGB Radiation Oncology Site Locator: [https://johnathan-zeng.github.io/romgblocator/](https://johnathan-zeng.github.io/romgblocator/)

The landing page is [index.html](./index.html), which links to both.

## Using On a Phone

Open the GitHub Pages URL for this site in your phone browser, then add it to your home screen so it behaves like a quick-launch app.

## Add to Home Screen on iPhone

1. Open the site in Safari.
2. Tap the Share button.
3. Scroll down and tap `Add to Home Screen`.
4. Rename it if you want.
5. Tap `Add`.

The shortcut will appear on your home screen and can open directly to the page you saved.

## Add to Home Screen on Android

1. Open the site in Chrome.
2. Tap the three-dot menu.
3. Tap `Add to Home screen` or `Install app`.
4. Rename it if you want.
5. Tap `Add` or `Install`.

The shortcut will appear on your home screen and can open directly to the page you saved.

## Tip

If you want separate icons for MGH and BWH, open each page directly before adding it:

- `.../mgh.html`
- `.../bwh.html`

If you want one shared launcher, add the main landing page instead:

- `.../`
